# 📦 Gestor de Inventario

Una aplicación de escritorio sencilla y eficiente para gestionar el inventario de productos. Permite llevar un control del stock de manera visual y rápida.

## ✨ Características
* **Gestión de Productos:** Crear, leer, actualizar y eliminar (CRUD) productos del sistema.
* **Control de Stock:** Botones dedicados para ingresar o retirar cantidades específicas de stock de manera segura.
* **Base de Datos Local:** Toda la información se guarda automáticamente de forma local.
* **Interfaz Gráfica:** Interfaz intuitiva y fácil de usar.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Java
* **Interfaz Gráfica:** Java Swing
* **Base de Datos:** SQLite (mediante JDBC)

## 🚀 Cómo ejecutarlo
No necesitas configurar un servidor de base de datos pesado. El programa crea su propia base de datos automáticamente la primera vez que se ejecuta.
